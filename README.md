# Application Stations autour de moi

## Description
Cette application affiche les stations proches de l'utilisateur en utilisant la géolocalisation du navigateur.

## Installation
1. Copier tous les fichiers dans un dossier local.
2. Ouvrir `index.html` dans un navigateur moderne.
3. Cliquer sur "Trouver les stations autour de moi".

## Configuration
- Les données de stations sont actuellement dans `app.js` (variable `stationsData`).
- Pour ajouter vos propres stations, modifier ce tableau avec les coordonnées réelles.

## Notes
- Fonctionne uniquement sur HTTPS ou localhost pour la géolocalisation.
